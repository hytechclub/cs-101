Boolean Operators - Exercises

Complete the following exercises.

Boolean Evaluation

Part 1

What is the boolean result of the following statements?

Determine what you think the answer is, and then write code to check your answer. Example: 25 == 25 -> true

  1. "Random String" != "Random String"
  2. 15 > 7
  3. 27 <= 152
  4. "My Name Is Jordan" == "my name is jordan"
  5. 35 >= 35
  6. 3.1254798 <= 5
  7. 9001 < 9001

Part 2

What is the FINAL boolean result for the following conditional? Example ( true && true ) -> true

  1. (false || false)
  2. (true || false)
  3. (true && false)
  4. !(true && false)
  5. (false || (true && true))
  6. (true && (true && false))
  7. ((false || !(true || false)) && !false)
  8. (false && (false || true))
  9. (3.1254798 <= 5 && 9001 < 9001)
  10. ("Random String" != "Random String" || (27 <= 152 && 35 >= 35)
  11. (false || (true && (false || true) || !(true && false) || (true && false)))
